데이터
변수명 변수타입 설명
userList
id String 사용자 아이디
name String 사용자 이름
displayId String 사용자 별명
provider String 가입경로
level Integer 레벨
point Integer 포인트
registerDate String 등록날짜
lastAccessDate String 마지막 접속 날짜
profileImgUrl String 프로필 이미지 주소
userStatus String (Y or N) 사용자 계정상태 (Y : 사용, N : 사용불가)
