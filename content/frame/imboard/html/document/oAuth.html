7. OAuth 사용법

임보드는 Facebook, Twitter, Google의 OAuth를 활용하여 로그인을 할 수 있습니다. 이 기능을 사용하기 위해서는 각 프로바이더의 AppId와 Secret key가 필요합니다.

/resources/properties/config.json 파일을 열어서 OAuth부분에 각 프로바이더의 AppId와 Secret key를 입력합니다.

"OAuth": {
	"facebook": {
	    "appId": "appId",
	    "secret": "secret"
	},
	"twitter": {
	    "appId": "appId",
	    "secret": "secret"
	},
	"google": {
	    "appId": "appId",
	    "secret": "secret"
	}
}

각 프로바이더 로그인버튼 구현방법

<a href="/auth/twitter">트위터</a>
