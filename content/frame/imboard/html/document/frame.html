2. 프레임

2-1. 프레임구조

프레임은 반드시 아래와 같은 구조를 가지고 있어야 합니다. 프레임 이름 폴더 하위에 css, html, js폴더가 위치하고 html폴더 하위에는 반드시 index.html 파일이 있어야 하며 이 파일은 웹페이지의 첫번째 페이지가 됩니다.

  • 프레임이름
    • css
    • html
      • index.html
    • js

2-2. index.html

index.html의 기본 구조입니다. 다섯가지의 js파일은 반드시 import되어야 합니다.

<!DOCTYPE html>
<html lang="ko">
<head>
<meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=EDGE"/>

<meta name="Title" content="title">
<meta name="author" content="author"/>
<meta name="reply-to" content="email"/>
<meta name="description" content="description"/>

<title>title</title>

<link rel="stylesheet" type="text/css" href="#{path.module}/imboard/css/imboard-ui-1.0.css" />

<script src="##{path.lib}/jquery/jquery-1.10.2.js"></script>
<script src="##{path.lib}/handlebars/handlebars-v1.3.0.js"></script>
<script src="##{path.module}/imboard/js/imboard-api-1.0.js"></script>
<script src="##{path.module}/imboard/js/imboard-ui-1.0.js"></script>
<script src="##{path.module}/imboard/js/handlebar-helper-1.0.js"></script>

</head>
<body>
</body>
</html>
