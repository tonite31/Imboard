Imboard

임보드는 웹페이지 개발을 간편하게 할 수 있는 nodejs기반 플랫폼입니다. HTML5의 속성을 지정하는것만으로 레이아웃을 구성하고 데이터를 화면에 조합할 수 있습니다.

다운로드

임보드의 최신버전은 zip과 tar.gz형태로 배포됩니다.

개발중인버전은 https://github.com/tonite31/Imboard/tree/development 에서 다운로드 받을 수 있습니다.

만약 npm을 사용할 수 있다면 커맨드라인에서 npm install imboard 명령어를 입력하면 간편하게 설치할 수 있습니다.

설치하기

사전준비

1. 웹서버

임보드를 설치할 호스팅 서버 또는 개인 서버가 필요합니다.

2. nodejs & MySQL

임보드는 nodejs기반 플랫폼이며 데이터베이스는 MySQL을 사용합니다. 최신버전의 nodejs와 MySQL을 다운로드하여 설치해주시기 바랍니다.

3. 필요한 nodejs 모듈 설치

임보드의 루트 디렉토리에서 npm install 명령어를 수행합니다.

4. 설정파일 수정

resources/properties/config.json 파일을 열어서 자신의 환경에 맞게 설정값을 입력합니다. frame과 jdbc 설정값은 필수적으로 입력해야합니다.

5. 임보드 어플리케이션 실행

임보드의 루트 디렉토리에서 node app.js 명령어를 수행합니다.

6. 웹페이지 접속

웹브라우저를 열어서 주소창에 http://localhost:3000/ 을 입력합니다.
