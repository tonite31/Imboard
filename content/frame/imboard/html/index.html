Imboard

임보드는 쉽고 빠르게 웹페이지를 개발할 수 있는 nodejs기반 플랫폼입니다.

간편한 방법으로 웹페이지를 개발할 수 있고, 미리 만들어진 프레임을 적용하면 몇 분만에 웹페이지를 개설할수도 있습니다.

다운로드

임보드의 최신버전은 zip과 tar.gz형태로 배포됩니다.

개발중인버전은 https://github.com/tonite31/Imboard/tree/development 에서 다운로드 받을 수 있습니다.

만약 npm을 사용할 수 있다면 커맨드라인에서 npm install imboard 명령어를 입력하면 간편하게 설치할 수 있습니다.

설치하기

사전준비

1. 웹서버

임보드를 설치할 호스팅 서버 또는 개인 서버가 필요합니다.

2. nodejs & MySQL

임보드는 nodejs기반 플랫폼이며 데이터베이스는 MySQL을 사용합니다. 최신버전의 nodejs와 MySQL을 다운로드하여 설치해주시기 바랍니다.



다운로드 및 설치

1. 임보드 다운로드

홈페이지에서 임보드를 다운로드 하거나 커맨드라인에서 npm install imboard를 입력해서 설치합니다.

2. 필요한 nodejs 모듈 설치

임보드의 루트 디렉토리에서 npm install 명령어를 수행합니다.

3. 데이터베이스 및 사용자 생성

설치한 MySQL에 접속한 뒤, 사용자와 데이터베이스를 생성합니다.

4. 데이터베이스 접속정보 설정

/resources/properties/config.json 파일을 열어서 jdbc부분에 데이터베이스 접속정보를 입력합니다.

5. 초기데이터입력

임보드의 루트 디렉토리에서 node install.js 명령어를 수행합니다. 4번에서 설정한 데이터베이스로 초기데이터가 입력되고 임보드 설치작업이 끝납니다.



임보드 어플리케이션 실행하기

1. frame 설정

/resources/properties/config.json 파일을 열어서 frame부분에 설정할 프레임 이름을 입력합니다. 프레임 이름은 /content/frame/ 디렉토리 하위의 폴더 이름입니다. 기본은 blog입니다.

2. 임보드 어플리케이션 실행

임보드의 루트 디렉토리에서 node app.js 명령어를 수행합니다.

3. 웹페이지 접속

웹브라우저를 열어서 주소입력부분에 http://localhost:3000/ 을 입력합니다.

4. 관리자페이지 접속

웹브라우저의 주소입력부분에 http://localhost:3000/settings 를 입력합니다. 관리자 계정으로 로그인하면 관리자페이지로 접속할 수 있습니다. 초기 관리자 계정은 admin/admin입니다.
