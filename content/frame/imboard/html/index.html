Imboard

임보드는 다양한 웹페이지 또는 웹서비스를 개발할 수 있는 nodejs기반 게시판 플랫폼입니다.

다운로드
최신버전 v0.0.1

GitHub
개발중인버전

nodejs, HTML, JS, CSS기반으로 웹페이지를 제작합니다.

당연하게도 임보드로 제작하는 웹페이지는 HTML과 JS, CSS만으로 작성할 수 있습니다. 물론 jQuery와 화면에 데이터를 조합하는데 도움을 주는 Handlebars라는 자바스크립트 라이브러리가 반드시 필요하지만 전혀 어렵지 않습니다.

프론트엔드에 집중하십시오. 백엔드는 이미 준비되어있습니다.

아름다운 화면을 디자인하는데 집중하세요! 필요한 대부분의 서버기능, 화면제작에 필요하지만 구현이 까다롭거나 단순하고 지루한 반복작업들을 최소화할 수 있는 다양한 컴포넌트들이 제공됩니다. 데이터를 관리하는 관리자페이지는 덤입니다. 못믿으시겠다면 문서페이지를 확인해보세요!
