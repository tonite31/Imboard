시작하기

임보드를 다운로드받아 웹서버에 설치하고 샘플 웹페이지를 구동합니다.

다운로드

임보드의 최신버전이 .zip파일과 .tar.gz파일로 제공됩니다.

현재 개발중인 버전은 https://github.com/tonite31/imboard에서 제공되고 있습니다.

다운로드
Imboard_0.0.zip

다운로드
Imboard_0.0.tar.gz

설치

임보드의 서버모듈은 nodejs기반으로 제작되었으며 MySQL을 기본 데이터베이스로 사용합니다. 임보드를 설치하기 전 nodejs와 MySQL을 설치해주시기 바랍니다.

설치하려는곳에 다운로드 받은 Imboard.zip 또는 Imboard.tar.gz압축을 해제합니다.

커맨드라인에서 npm install을 실행해서 필요한 모듈을 다운로드받습니다.

set database를 실행해서 database를 설정합니다. 여기서 database이름, 유저정보 입력받고 ddl, dml실행

환경설정

/resources/properties/config.json 파일을 열어서 프레임정보를 입력하고 설치단계에서 설정했던 데이터베이스정보를 입력합니다.

프레임은 웹페이지 화면들의 집합으로 워드프레스의 테마와 성격이 비슷합니다.

샘플 웹페이지 구동

node app.js 명령어를 실행해서 서버를 구동합니다.

웹브라우저를 실행하고 주소창에 http://localhost:3000/를 입력합니다.

샘플 프레임이 적용된 웹페이지가 열렸습니다!
