1. 개요

임보드는 웹페이지를 구성하는 프레임을 제작하는것부터 시작합니다. 프레임은 HTML, JS, CSS, 그리고 이미지등의 각종 멀티미디어 파일로 구성됩니다.

1. data-bind 모듈

1-1. 개요

data-bind 모듈은 HTML 태그에 data-bind, data-param, data-template-id 속성을 지정해서 사용할 수 있습니다.

불러온 데이터는 Handlebars 문법으로 화면에 조합할 수 있습니다.

<div data-bind="모듈명" data-param='{"key" : "value"}' data-template-id="핸들바 템플릿 아이디"></div>

2. API

2-1. 개요

API는 서버로 요청을 보내고 응답을 받는 함수들과 각종 데이터를 제어할 수 있는 객체로 구성됩니다. 반드시 와 /content/module/imboard/js/imboard-api-1.0.js 자바스크립트 파일을 HTML파일에 import해야 사용할 수 있습니다.

2-2. Query

URL파라미터를 key-value 형식으로 가져옵니다.

2-3. Form

폼 submit이 발생했을때 폼 내부의 name 또는 data-name속성을 가지고 있는 엘리먼트의 값을 파라미터로 모아주고 HTML5 validation을 수행합니다.

2-4. 페이지전환

페이지의 일부를 전환하거나 페이지 전체를 다른 페이지로 전환합니다.

URL파라미터에 piece라는 키로 할당된 값에 해당하는 html파일을 아래와 같이 data-fragment속성이 지정된 태그의 innerHTML로 보여줍니다.
<div data-fragment="piece"></div>
아래와 같이 .page로 끝나는 URL이 입력된 경우 해당경로에 위치한 html파일로 페이지 전체를 전환합니다. 예제 URL의 경우 /html/test/test.html파일.
http://localhost:3000/test/test.page
