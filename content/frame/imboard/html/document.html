1. data-bind 모듈

1-1. 개요

data-bind 모듈은 예제와 같이 HTML 태그에 data-bind 속성과 data-param(optional) 속성을 지정해서 사용할 수 있습니다. 불러온 데이터는 Handlebars 문법으로 화면에 조합할 수 있습니다.

<div data-bind="모듈명" data-param='{"key" : "value"}'></div>

1-2. Board 모듈

<script id="boardListTemplate" type="text/x-handlebars-template">
{{#if boardList}}
<table>
	<colgroup>
		<col/>
	</colgroup>
{{#each boardList}}
{{/each}}
</table>
{{/if}}
</script>
<div data-bind="boardList" data-template-id="boardListTemplate"></div>

게시글을 불러오는 모듈입니다. 단순하게 모든 게시글 목록을 불러오거나 페이징처리된 게시판 형태로 게시글 목록을 불러옵니다.

2. API

2-1. 개요

API는 서버로 요청을 보내고 응답을 받는 함수들과 화면의 데이터를 손쉽게 제어할 수 있는 객체로 구성됩니다. 반드시 /content/module/imboard/js/imboard-ui-1.0.js와 /content/module/imboard/js/imboard-api-1.0.js 자바스크립트 파일을 HTML파일에 import해야 사용할 수 있습니다.

2-2. Querystring

자바스크립트 코드에서 URL파라미터에 손쉽게 접근할 수 있는 객체를 제공합니다.

http://localhost:3000?id=user&name=username 
$(document).ready(function()
{
	var id = $.query.id;
	var name = $.query.name;
	
	console.log("id : ", id);
	console.log("name : ", name);
});

2-4. 페이지전환

페이지의 일부를 전환하거나 페이지 전체를 다른 페이지로 전환합니다.

URL파라미터에 piece라는 키로 할당된 값에 해당하는 html파일을 아래와 같이 data-fragment속성이 지정된 태그의 innerHTML로 보여줍니다.
<div data-fragment="piece"></div>
아래와 같이 .page로 끝나는 URL이 입력된 경우 해당경로에 위치한 html파일로 페이지 전체를 전환합니다. 예제 URL의 경우 /html/test/test.html파일.
http://localhost:3000/test/test.page
