문서

임보드로 화면을 만들때 사용할 수 있는 각종 UI컴포넌트와 데이터 바인드 모듈, 서버요청 API에 대해 설명합니다.

1. UI컴포넌트

1-1. 개요

임보드의 UI컴포넌트를 사용하기 위해서는 /content/module/imboard/js/imboard-ui-1.0.js 자바스크립트 파일과 /content/module/imboard/css/imboard-ui-1.0.css 스타일시트파일을 HTML파일에 import해야 사용할 수 있습니다.

1-2. 페이지전환

페이지의 일부를 전환하거나 페이지 전체를 다른 페이지로 전환합니다.

URL파라미터에 piece라는 키로 할당된 값에 해당하는 html파일을 아래와 같이 data-component와 data-key속성이 지정된 태그의 innerHTML로 보여줍니다.
<div data-component="piece" data-key="piece"></div>
아래와 같이 .page로 끝나는 URL이 입력된 경우 해당경로에 위치한 html파일로 페이지 전체를 전환합니다. 예제 URL의 경우 /html/test/test.html파일.
http://localhost:3000/test/test.page
